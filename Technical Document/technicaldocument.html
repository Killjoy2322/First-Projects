Basic HTML document

DOCTYPE HTML HTML tag Head tag Body tag Footer tag

DOCTYPE HTML

The declaration is not an HTML tag. It is an "information" to the browser about what document type to expect.

  • <!DOCTYPE.html>

HTML tag

The tag represents the root of an HTML document. It is the container for all other HTML elements except for the !DOCTYPE tag. You should always include the lang attribute inside the tag, to declare the language of the Web page. This is meant to assist search engines and browsers.

  • <!DOCTYPE.html>
  • <html> and </html>

Head tag

It is a container for metadata or data about data and is placed between the html tag and the body tag. Metadata is data about the HTML document. Metadata is not displayed. Metadata typically define the document title, character set, styles, scripts, and other meta information. The following elements can go inside the head element and the ones without closing tags are called empty tags:

  • <head>
  • <title> and </title>
  • <style> and </style>
  • <base>
  • <link>
  • <meta>
  • <script> and </script>
  • <noscript> and </noscript>
  • </head>

Body tag

The body element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

  • <h1> and </h1> through <h6> and </h6>
  • <p> and </p>
  • <img>
  • <a>
  • <table> and </table>
  • <ol> and </ol> or <ul> and <ul>
  • etc.

Footer tag

This tag defines a footer for a document or section. It goes at the end of the document.

  • authorship information
  • copyright information
  • contact information
  • address information which should be in an <address> and </address> tag.
  • site map
  • back to top links
  • related documents
